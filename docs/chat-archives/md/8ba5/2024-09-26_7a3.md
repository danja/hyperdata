# Federated Wiki (FedWiki) Syntax Cheat Sheet

## Basic Formatting

- **Paragraphs**: Separate with blank lines
- **Line Breaks**: Use `\\` at the end of a line
- **Bold**: `**bold text**`
- **Italic**: `//italic text//`
- **Strikethrough**: `~~strikethrough text~~`

## Headers

- `# Level 1 Header`
- `## Level 2 Header`
- `### Level 3 Header`

## Lists

### Unordered Lists
```
* Item 1
* Item 2
  * Subitem 2.1
  * Subitem 2.2
```

### Ordered Lists
```
1. First item
2. Second item
   1. Subitem 2.1
   2. Subitem 2.2
```

## Links

- Internal Link: `[[Page Name]]`
- External Link: `http://example.com`
- Named External Link: `[Example](http://example.com)`

## Images

- `{{image.jpg}}`
- With alt text: `{{image.jpg|Alt text}}`

## Code

- Inline Code: `` `code` ``
- Code Block:
  ```
  ```language
  code goes here
  ```
  ```

## Blockquotes

```
> This is a blockquote
> It can span multiple lines
```

## Horizontal Rule

`----`

## Tables

```
|Header 1|Header 2|
|--------|--------|
|Cell 1  |Cell 2  |
|Cell 3  |Cell 4  |
```

## Plugins

- Calculator: `{{{calc 2+2}}}`
- Roster: `{{{roster}}}`
- Chart: `{{{chart}}}`
- (Note: Plugin syntax may vary)

## Comments

```
<!-- This is a comment -->
```

Remember, FedWiki uses a slightly modified Markdown syntax, so some standard Markdown elements might behave differently.
