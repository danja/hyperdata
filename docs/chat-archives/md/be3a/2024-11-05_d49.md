 Command `git submodule` or more explicitly:
```bash
git submodule status
```

This shows:
- Hash of the currently checked out commit
- Submodule path
- (branch name) if available
- Status indicators: 
  - '-' not initialized
  - '+' different version checked out
  - ' ' synchronized

For more detail:
```bash
git config --file .gitmodules --list
```
