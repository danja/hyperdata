Solution:
Check repository ownership using:
```bash
ls -la /path/to/repository
```
For more detailed ownership info including git-specific data:
```bash
ls -la /path/to/repository/.git
stat /path/to/repository
```

Key output fields:
- User (owner)
- Group
- Permissions
