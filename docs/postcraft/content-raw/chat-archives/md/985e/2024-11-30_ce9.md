 For XMPP persistence in a container setup, use a volume mount to store prosody's data:

```yaml
services:
  xmpp:
    image: prosody/prosody
    volumes:
      - ./prosody-data:/var/lib/prosody
    ports:
      - "5222:5222"
```

This preserves user accounts, message history, and configurations across container restarts.
