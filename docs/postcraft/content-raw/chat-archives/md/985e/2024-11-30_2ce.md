 The node-xmpp-server package is outdated and incompatible with current Node.js versions. Let's use a modern alternative:

Try installing `@xmpp/server`:
```bash
npm install @xmpp/server
```

Then update xmpp-server.js:
```javascript
const { server } = require('@xmpp/server')

const xmpp = server({
  domain: 'localhost',
  port: 5222,
})

xmpp.start()
  .catch(console.error)
```
