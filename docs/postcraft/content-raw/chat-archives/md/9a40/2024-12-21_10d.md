 The `twitter:card` meta tag specifically controls how your link preview appears on Twitter. 

`summary_large_image` displays a prominent, large image above your title and description text, spanning the full width of the tweet. This format is more visually engaging than the default `summary` card which shows a smaller thumbnail image to the left.

Available card types:
- `summary`: Small square image + text
- `summary_large_image`: Large banner image + text
- `player`: For video/audio content
- `app`: For mobile app links
