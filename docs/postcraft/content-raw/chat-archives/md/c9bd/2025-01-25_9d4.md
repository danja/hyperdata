 Here's a regex pattern for matching strings that:
1. Start with "#:"
2. Are followed by alphanumeric characters 
3. End at the next whitespace

```
#:[a-zA-Z0-9]+
```

This will match strings like:
- "#:abc123"
- "#:XYZ789"
- "#:test"

But won't match:
- "#abc" (missing colon)
- "#:!" (contains non-alphanumeric)
- "abc123" (doesn't start with #:)
