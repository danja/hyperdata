 Orphan modules occur because your CSS files aren't properly linked to the entry point. They exist in your project but aren't connected to the dependency graph.

To fix:
1. Either import CSS directly in app.js:
```javascript
import '../css/styles.css';
import '../css/form-styles.css';
```

2. Or define them as entry points in webpack config:
```javascript
entry: {
    main: './src/js/app.js',
    styles: './src/css/styles.css',
    formStyles: './src/css/form-styles.css'
}
```
